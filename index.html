Aadhar is a rails engine to support basic token-based authentication.

Add following lines in you Gemfile

gem 'aadhar'

Execute following commands to install 'aadhar' engine

rake aadhar:install:migrations
rake db:migrate

After installation following APIs will be available

1. Registration
2. Login
3. Logout
4. Reset password
