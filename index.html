Aadhar

Download this project as a .zip file Download this project as a tar.gz file

This is Rails engine to support basic token-based authentication.

Add following line in you Gemfile

gem 'aadhar'

Execute following command to instal 'aadhar' engine

rake aadhar:install:migrations
rake db:migrate

After installation following APIs will be available 1. Registration 2. Login 3. Logout 4. Reset password
